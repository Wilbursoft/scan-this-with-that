Niche Markets Study

Overview

Niche markets are potential markets, a platform or system, that have an extensible plugin eco system where TPM plugins or integrations could be released.

These TPM plugins would have hooks into the platform and provide call back into TSM cloud provide in order to provide some security value add to the platform.

Users can the use the TPM platform to pick and choose between

Criteria

·      Ease of Integration: How easy would it be to produce and integration or plugin for the platform.

·      Active Community: Is there an active community to feedback,  or maintain the plugin?

·      Security Need: How compelling is the security value proposition for the platform?

·      Existing Offerings: Are there existing single vendor of free offerings? What are there down falls? Are the open sourced? Could TPM be added in?

·      Rival Platform: Does a rival platform have security offering that would could be cloned and enhanced via TPM?

·      Eco System Buzz: Is there signs of activity in the eco system a discussion forum where specific needs can be gleaned?

·      Size: How many users of the platform are there. How many of those actively browse the plugin store?
